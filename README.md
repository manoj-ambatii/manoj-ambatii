## 🚀 Featured Projects

### 📈 [StockPuls – AI-Powered Investment Tracker](https://github.com/manoj-ambatii/StockPuls)

> **React (TypeScript) · Node.js · Express · MySQL · Tailwind CSS · Gemini API · Yahoo Finance API**

Full-stack investment tracking platform for Indian stocks (NSE/BSE) with portfolio management, watchlists, and real-time quotes.

- 🤖 Google Gemini AI generates **BUY/HOLD/SELL insights** with confidence scores, cached in MySQL to reduce redundant API calls
- 📊 Live quotes via Yahoo Finance (crumb-cookie workflow) — 11-row income statements, financial ratios & 7 fundamentals sections per stock
- ⚡ React Context with multi-dimensional per-symbol caching (quotes, charts, fundamentals) eliminating redundant API calls
- 🔐 JWT auth · React Hook Form + Zod validation · context-aware Axios interceptors for 401 handling
- 🚀 React.lazy + Suspense for code splitting — frontend on Render, MySQL on Railway

### 🚢 [Voltusfreight – Freight Forwarding Platform](https://github.com/manoj-ambatii) *(Voltuswave Technologies)*

> **React (TypeScript) · Node.js · MySQL · AWS · AWS Cognito · Docker**

Multi-tenant freight forwarding platform enabling businesses to manage logistics operations with isolated, tenant-specific data.

- 🔐 End-to-end user management with **AWS Cognito** — signup, login, role-based authorization, token management
- 📊 Real-time admin dashboard with user activity tracking, session durations & behavioural analytics
- 🐳 Containerized with **Docker**, deployed on **AWS** for high availability and reliability
- ⚙️ CI/CD pipelines with **GitHub Actions** — automated build, test, and deployment workflows
